# ebsant

Take snapshot of AWS EC2's volume

## Comands

1. Execute

    ~~~
    python /path/to/main.py -r {REGION_NAME} -i {AWS_ACCESS_KEY_ID} -k {AWS_SECRET_ACCESS_KEY}
    ~~~

2. Show help

    ~~~
    python /path/to/main.py -h
    ~~~

3. Show version info

    ~~~
    python /path/to/main.py -v
    ~~~
